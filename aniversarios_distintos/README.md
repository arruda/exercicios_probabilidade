# Aniversários Distintos

Ao selecionar R pessoas, qual a probabilidade que todas elas façam aniversário em dias distintos.
Considerando que a probabilidade de cada dia seja igual.


## Formula da resolução

365!/(365-r)! * (365)^r

Onde r é o numero de pessoas.
